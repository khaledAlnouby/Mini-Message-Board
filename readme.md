# Mini Message Board

A simple Express.js application that allows users to post messages.

## Features

- View all messages
- Add new messages
- Server-side rendering with EJS
- Express routing
- Form handling with POST requests

## Tech Stack

- Node.js
- Express.js
- EJS

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/mini-message-board.git
cd mini-message-board
npm install
npm start
```

Visit:

```
http://localhost:3000
```
